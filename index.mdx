One command surfaces what the internet actually thinks about any topic in the last 30 days.

## What last30days does

Reddit upvotes. X likes. YouTube transcripts. TikTok engagement. Polymarket odds backed by real money. last30days searches all of it in parallel, scores by what real people engage with, and synthesizes it into one brief.

## Get started

<Steps>
  <Step title="Install the skill">
    Install on your AI platform of choice. Claude Code is the fastest path:

    ```bash
    /plugin marketplace add mvanhorn/last30days-skill
    ```
  </Step>
  <Step title="Run your first research query">
    Type a topic — person, product, event, or "X vs Y" comparison:

    ```
    /last30days Sam Altman
    /last30days Claude Code vs Codex
    /last30days Universal Epic Universe
    ```
  </Step>
  <Step title="Unlock more sources (optional)">
    The first-run setup wizard takes 30 seconds and adds X/Twitter (via browser cookies) and YouTube (via yt-dlp). TikTok and Instagram are available with a free ScrapeCreators key.
  </Step>
</Steps>
