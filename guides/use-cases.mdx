last30days works for any topic where you want to know what people are actually saying right now — not what editors decided to publish. Below are the most common use cases, with real examples of what the engine surfaces.

<AccordionGroup>
  <Accordion title="Before a meeting">
    **Query:** `/last30days Peter Steinberger`

    You have a meeting tomorrow. You Google them. You get their LinkedIn from 2023.

    last30days gives you what they're actually doing this month: joined OpenAI's Codex team, fighting Anthropic's ban on third-party agents, 23 PRs merged at 85% merge rate on GitHub, building LobsterOS for cross-device agent control. And then there's r/ClaudeCode with a 227-upvote thread: "Ever since OpenClaw released, it was widely known that if you run it through anything other than the API, you were gonna get banned eventually." That's not on LinkedIn.

    Add `--github-user=steipete` and you also get PR velocity, top repos with star counts, and release notes for what shipped this month.

    **Other meeting research queries:**
    ```bash
    /last30days Sam Altman
    /last30days Dario Amodei
    /last30days {company name} before a sales call
    ```
  </Accordion>

    /last30days {company} funding announcement

## Source coverage by use case

Different topics have different signal distribution across platforms. Here's where the strongest signal typically lives:

| Use case | Strongest sources |
|---|---|
| Person / meeting prep | X posts, GitHub (person-mode), YouTube transcripts |
| Music / entertainment | Reddit, YouTube reviews, TikTok, Polymarket |
| Tech product comparison | Reddit, GitHub (live stars), X, HN |
| Current events / geopolitics | X, YouTube, Polymarket odds |
| Trip planning | Reddit, YouTube |
| Prompting techniques | Reddit, X, YouTube tutorials |

<Tip>
  After any research run, the AI session has absorbed everything the engine surfaced. You can ask follow-up questions, request deeper dives, or have it write prompts and emails grounded in what's actually happening — without running another search.
</Tip>

## Zero-config sources

Some sources work immediately with no setup. You can run any of the use cases above and get results from:

- **Reddit** — threads and top comments with upvote counts via public JSON
- **Hacker News** — stories and comments via the Algolia API
- **Polymarket** — prediction market odds via the Gamma API
- **GitHub** — live star counts and issue data (enhanced if you have the `gh` CLI installed)

X/Twitter, YouTube, TikTok, and Instagram require a one-time setup that takes about 30 seconds. Run `/last30days` once and the setup wizard walks you through it.
